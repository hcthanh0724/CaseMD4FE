jQuery Validation v1.17.0 - Pro Component

If you have required fields in forms, the best way to add validation for them is through the jQuery Validation tool. We have imported it into the Material Dashboard .Net Core 2.2 Stater library and changed its colors.

For more information please check Full Github Documentation.
